# Sherlock - Username Search

## Tool Overview
Purpose: Find usernames across 400+ social networks
Location: /usr/bin/sherlock

## When to Use
- Finding a person's online presence
- Username reconnaissance

## How to Use It

Basic Search:
sherlock username

Multiple Usernames:
sherlock user1 user2 user3

Timeout:
sherlock --timeout 5 username

CSV Output:
sherlock --csv username

# ghunt - Google OSINT

## Tool Overview
Purpose: Google account reconnaissance tool
Location: /home/kali/.local/bin/ghunt

## When to Use
- Investigating Google accounts
- Finding Google services linked to emails

## How to Use It

Login First:
ghunt login

Check Email:
ghunt email email@example.com

Check Username:
ghunt username

# recon-ng - Full OSINT Framework

## Tool Overview
Purpose: Full-featured reconnaissance framework with modules
Location: /usr/local/bin/recon-ng

## When to Use
- Comprehensive OSINT investigations
- Automated reconnaissance workflows
- Multiple data source correlation

## How to Use It

Start recon-ng:
recon-ng

Basic Commands:
workspace create target
marketplace search
marketplace install hackertarget
modules load hackertarget
options set SOURCE example.com
run

## Key Modules
- hackertarget - Subdomain enumeration
- shodan - Shodan.io integration
- whois - WHOIS lookups
- bing - Bing search
- google - Google search

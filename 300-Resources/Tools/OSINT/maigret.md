# maigret - Username OSINT

## Tool Overview
Purpose: Advanced username search across many sites
Location: /home/kali/.local/bin/maigret

## When to Use
- Deep username reconnaissance
- Finding social media profiles
- Identity correlation

## How to Use It

Basic Search:
maigret username

Save Results:
maigret username --folder output/

JSON Output:
maigret username --json output.json

Top Sites Only:
maigret username --top-sites

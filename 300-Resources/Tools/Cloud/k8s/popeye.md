# Popeye - Kubernetes Cluster Sanitizer

## Tool Overview
Purpose: Scan Kubernetes clusters for best practices
Location: /usr/local/bin/popeye

## When to Use
- Cluster health checks
- Best practice validation
- Resource optimization

## How to Use It

Scan cluster:
popeye

Save report:
popeye --out report.json

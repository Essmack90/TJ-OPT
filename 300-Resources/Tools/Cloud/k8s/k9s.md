# k9s - Kubernetes Terminal UI

## Tool Overview
Purpose: Terminal-based UI for managing Kubernetes
Location: /usr/local/bin/k9s

## When to Use
- Kubernetes cluster management
- Real-time monitoring
- Resource inspection

## How to Use It

Start k9s:
k9s

Commands:
:namespace - Switch namespace
:pods - View pods
:describe - Describe resource
:logs - View logs
q - Quit

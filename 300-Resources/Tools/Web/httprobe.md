# httprobe - HTTP Probe

## Tool Overview
**Purpose:** Probe for working HTTP/HTTPS servers
**Location:** `which httprobe`

## When to Use
- Filtering live hosts from subdomain list
- Checking which hosts respond

## How to Use It
cat subdomains.txt | httprobe

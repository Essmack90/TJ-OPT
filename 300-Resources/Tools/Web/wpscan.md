# wpscan - WordPress Vulnerability Scanner

## Tool Overview
**Purpose:** WordPress specific enumeration and vulnerability scanning
**Location:** `which wpscan`

## When to Use
- WordPress sites only
- Finding vulnerable plugins/themes
- User enumeration

## How to Use It

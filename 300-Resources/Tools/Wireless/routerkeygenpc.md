# Router Keygen - Default WiFi Key Generator

## Overview
Generates default WPA keys for vulnerable routers based on SSID/BSSID

## Location
/usr/bin/routerkeygen-cli

## Usage

### Generate Keys from BSSID
routerkeygen-cli -m AA:BB:CC:DD:EE:FF

### Generate with SSID
routerkeygen-cli -s "MyWiFi" -m AA:BB:CC:DD:EE:FF

### Quiet Mode
routerkeygen-cli -q -m AA:BB:CC:DD:EE:FF

### GUI Mode
routerkeygen-gui

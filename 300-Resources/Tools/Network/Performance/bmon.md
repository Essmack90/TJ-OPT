# bmon - Bandwidth Monitor

## Overview
Real-time bandwidth monitoring

## Location
/usr/bin/bmon

## Basic Usage
bmon
bmon -p eth0
bmon -r 2

## Common Options
-p : Interface
-r : Refresh rate
--htype : Graph type

## Related Tools
nethogs, iftop

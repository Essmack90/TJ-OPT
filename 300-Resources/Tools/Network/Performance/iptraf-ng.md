# iptraf-ng - Network Statistics

## Overview
Interactive network monitoring tool

## Location
/usr/sbin/iptraf-ng

## Basic Usage
sudo iptraf-ng
sudo iptraf-ng -i eth0
sudo iptraf-ng -s eth0

## Common Options
-i : IP traffic monitor
-s : TCP/UDP service monitor
-g : General interface statistics

## Related Tools
nethogs, iftop

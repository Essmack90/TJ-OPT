# nethogs - Per-Process Network Traffic

## Overview
Monitor network traffic per process

## Location
/usr/sbin/nethogs

## Basic Usage
sudo nethogs
sudo nethogs eth0

## Common Options
-d : Delay between updates
-t : Tracemode
-c : Number of updates

## Related Tools
iftop, iptraf-ng
